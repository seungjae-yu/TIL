# vscode

## <b>유용한 플러그인</b>

-   Jest / Jest Runner

    > jest를 사용하는 테스트 코드를 커맨드 입력없이 바로 Debug | Run 가능

    > test를 각각 돌려볼 수 있어 편하다.

-   Git Graph
    > 별도의 프로그램 없이 git tree 확인 가능

## <b>유용한 단축키</b>

-   사용하지 않는 import 문 제거

    > Alt + Shift + O (for Windows)

    > Option + Shift + O (for Mac)
