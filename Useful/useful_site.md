# 자주 사용하는 유용한 사이트

## <b>json2ts</b>

- http://json2ts.com/

  > json을 입력하면 interface로 뽑아준다.  
  > API response를 interface로 타입 가져올 때 매우 편하다.

## <b>xml to json</b>

- https://www.utilities-online.info/xmltojson

  > XML을 json으로 변환시켜준다.
